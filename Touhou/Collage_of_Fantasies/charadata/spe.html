■ＳＰＥのルール
  • ＳＰＥの宣言は原則として、それ自体でアクションフェイズは消費しない。
  • 特記がなければ発動は任意である。
  • 応用技同様、セッションごとの付け替えも可能。
  • ＳＰＥもキャラクターを表現する要素の一つである。
    そのキャラクターの異能・設定と絡められるようなものを選択することを推奨。

闘志
【更新】

■使用タイミング /
■使用制限 / １セッションに１回
■消費 / なし
■効果 /

※戦闘不能は起こらなかったものとして扱う。

●闘志状態：
ＳＰＥ【闘志】によって発生する、特殊な状態異常。
状態異常者はダメージ判定に＋１ｄのボーナスを得るが、
ＨＰを回復できない。
この効果は無効化されない。

■補足 /

  • 継続ラウンドを廃止。

■使用タイミング /
■使用制限 / なし
■消費 / なし
■効果 /

    応用技等との併用は不可能。
  • 本家と違い、任意発動可。

共鳴

■使用タイミング① /
■使用制限① / なし
■消費① / ＰＰ４点
■効果① / 現在アクションフェイズに参加している味方１人を対象として宣言可能。
以下の効果の内、任意のものをそれぞれ適用できる。
●対象に応用技を使用する
●対象に消費アイテムを使用する
●ＡＥ【コンセントレイション】を代理で宣言する（Lv5まで）
■補足① /

  • 連携バーストと違い禁止技はなし
  • 応用技は単発化しないが、その代わり上書きルールも適用される。

■使用タイミング② /
■使用制限② / なし
■消費② / ＰＰ４点と自身のメインアクションフェイズ
■効果② /

自身が未行動の場合、現行アクションフェイズで攻撃を行う味方１人を対象として宣言可能。
自身もアクションフェイズに参加し、同じ敵を対象として攻撃行動の宣言ができる。
（能動側の参加者が２人いる特殊なアクションフェイズとなる）

命中判定は２人とも行い、どちらか好きな判定結果を使用できる。
（クリティカル数などもそちらを参照する）
ダメージ計算は２人のダメージ判定の合計から敵の装甲値を差し引く。

なお、以下のアクションフェイズは同じ属性の攻撃でなければ共鳴できない。

  • 物理攻撃
  • 精神攻撃
  • 全体攻撃
  • 貫通攻撃
  • 自動命中
  • ヘッドショット攻撃
  • ブレイク攻撃
■補足② /

  • ロケットランチャー及び使い捨てランチャーを使った共鳴は不可。

パージ
【更新】

■使用タイミング /
■使用制限 / セッション中に１回
■消費 / なし
■効果① /

自身の全ての特殊武装を、このセッション中使用不可にする。
その後、以下の補正をそれぞれ適用する。
●このセッション中、【使用不可能になった武装による圧迫スペックの合計】分だけ、
　発動スペックと行動値ボーナスを得る。
●宣言ラウンド中、命中・回避判定にも同値のボーナスを獲得する。

■効果② /

ダメージ判定中に宣言した場合、この効果を追加で適用する。
そのダメージ判定で受けるＨＰダメージを、
【解放スペック×２】点まで軽減する（最低値１）。

■補足 /

  • アタッチメントはパージしなくて良い。
  • 行為判定中の宣言はダメージ判定限定。
    命中回避判定中の宣言などは不可。
  • カルマ【武器屋】による特殊武装も同様に使用不可能となるが、
    カルマによる武装装備はスペックの圧迫を行っていないため、
    各ボーナスも得られない。
